# remark-editor

一个基于语法树（mdast）的富文本 Markdown 编辑器。[在线示例](https://hxddev.github.io/remark-editor/)

## 安装

```bash
npm install --save remark-editor
```

## 使用

```jsx
import React, { Component } from "react";

import { Editor } from "remark-editor";

class App extends Component {
  render() {
    return <Editor />;
  }
}
```
